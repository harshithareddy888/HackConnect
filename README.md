# HackConnect Backend

A platform for hackathon team formation and collaboration.

## Features

- User authentication with JWT
- Profile management
- Team creation and management
- Task management
- Matching system

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Create a [.env](cci:7://file:///c:/Users/harshitha%20reddy/OneDrive/Documents/Projects/HackConnect/backend/.env:0:0-0:0) file based on `.env.example`
4. Start the development server: `npm run dev`

## API Documentation

Access the API documentation at `http://localhost:5000/api-docs`

## Testing

Run tests: `npm test`

## Production

To run in production:
1. Set `NODE_ENV=production`
2. Update environment variables
3. Run `npm start`
